Additional Information:
Join an agile software development team designing and building advanced, state of the art mission planning software for the B-2 bomber. The B-2 MPE (Mission Planning Environment) is updating the mission planning system for the B-2 bomber from the UNIX AFMSS system to use the JMPS (Joint Mission Planning System) framework that runs on Windows. This project is creating the UPC (Unique Planning Component) for the specialized needs of the B-2 bomber. The code is primarily C# and written in the Windows development environment.
