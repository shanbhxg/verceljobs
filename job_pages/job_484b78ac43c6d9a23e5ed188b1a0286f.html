Full Time | Seattle, WA

Lead the future direction for developer tools at Amazon. Amazon Web Services is looking for a Principal Technical Program Manager to define the strategy for, and then deliver on, improvements to the developer tools ecosystem across all of Amazon. This position is great for technical leaders with an eye for a great user experience, and can work across a large organization to deliver results. This is a highly visible role that will interact with all levels of employees and leadership, with a technical evangelist-like need for both coding and communication skills.
