Additional Information:
Planet is seeking a results-oriented, self-motivated Dev Ops systems engineer / developer who will be responsible for securing and maintaining an enterprise network and the underlying network infrastructure services and management platforms. The engineer will develop scripts and web-based interfaces to automate configuration changes, generate reports, correlate event data from network devices, and operate the enterprise Network Access Control (NAC) system. The engineer will provision and maintain Cent OS / Red Hat Linux servers and network management application software, both open source and COTS. The engineer will also administer Infoblox ID Grid appliances. This position is responsible for the operation and maintenance of network monitoring applications and infrastructure services and must ensure each change to the environment is clearly documented, to include creating and updating all related engineering documents, drawings, and IP addressing assignments.
